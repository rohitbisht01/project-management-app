# Team Project Management System

## Features

1. **Multiple Workspaces**: Users can create and manage multiple workspaces with different teams.
2. **Project Management**: Users can create, manage, and organize multiple projects within a workspace.
3. **Member Invitations**: Users can invite team members to a workspace via unique invite links.
4. **Task Management**: Users can create tasks and assign them to specific members for better collaboration.
5. **Admin Controls**: Workspace admins can assign admin privileges to specific members for better role management.
6. **Authentication & Authorization**: Secure user authentication using Passport.js with role-based access control.

## Tech Stack

### Frontend

- **React** – Component-based UI development
- **TypeScript** – Strongly typed JavaScript for better maintainability
- **TanStack Query** – Efficient state and server state management
- **Zustand** – Lightweight state management for global state
- **TailwindCSS** – Utility-first styling for a modern and responsive UI

### Backend

- **Node.js & Express.js** – Scalable backend API
- **MongoDB** – NoSQL database for flexible data storage
- **Passport.js** – Authentication and authorization
